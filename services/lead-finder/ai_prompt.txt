KROK 3.1 - OSTRA GRANICA: TYPE A vs TYPE B
=================================================

KLUCZOWE PYTANIE: "Czy firma wysyła WŁASNYCH LUDZI do prowadzenia imprezy?"

TYPE A jeśli TAK — firma ma PERSONEL (animatorzy, koordynatorzy, prowadzący program,
DJ, wodzirej), który przyjeżdża i prowadzi wydarzenie z udziałem publiczności.
Sygnały TYPE A bez explicite "DJ/wodzirej":
  ✓ "nasz zespół animatorów", "nasi koordynatorzy"
  ✓ "organizujemy od A do Z", "kompleksowa obsługa"
  ✓ "program integracyjny z prowadzącym"
  ✓ "agencja eventowa" + własna domena firmowa
  ✓ team building z programem aktywności prowadzonym przez firmę

TYPE B jeśli NIE — firma TYLKO:
  ✗ wynajmuje miejsce ("nasza sala", "nasz park", "nasz ośrodek")
  ✗ wynajmuje sprzęt lub konkretną atrakcję fizyczną
  ✗ organizuje imprezy ALE wyłącznie jako catering/logistyka bez personelu prowadzącego

SZARA STREFA — firmy oferujące "organizację imprez" z atrakcjami mieszanymi
(np. paintball + piknik + grill):
  → Jeśli w opisie pojawia się "animator", "prowadzący", "program z prowadzącym",
    "imprezy od A do Z z obsługą" → TYPE A
  → Jeśli to głównie "obiekt z paintballem i opcją grilla" → TYPE B
